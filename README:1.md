# IELTS Practice Resources Hub

A simple, organized collection of websites to help with IELTS preparation, sorted by skill (Listening, Reading, Writing, Speaking) plus general English improvement resources, vocabulary tools, and band score calculators.

## Features

- Sites organized by section: Listening, Reading, Writing, Speaking
- General skill-development resources (BBC, TED, Grammarly, etc.)
- Vocabulary & grammar tools
- Band score and progress tools
- Search/filter box to quickly find a site
- Simple, clean, responsive design

## Usage

Just open `index.html` in any browser, or visit the live site (if hosted via GitHub Pages / Netlify).

## Notes

- Some links may go stale over time. If you find a dead link, feel free to open an issue or PR.
- This was made as a personal project to keep track of useful IELTS prep sites.
